# StockSift 选股软件产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
StockSift 是一款面向个人投资者的 A 股桌面选股分析工具，专注于提供快速、高效的股票筛选和分析能力，帮助用户在短时间内发现投资机会。

### 1.2 目标用户
- 个人股票投资者
- 短线/波段交易者
- 价值投资用户
- 量化投资爱好者

### 1.3 核心价值
- **快速筛选**: 多维度条件组合，秒级筛选全市场股票
- **实时分析**: 技术指标实时计算，资金流向追踪
- **基本面分析**: 深度财务数据挖掘，价值投资指标评估
- **直观展示**: K线图表、数据表格、可视化面板
- **策略回测**: 验证选股策略的历史表现

## 2. 功能模块设计

### 2.1 股票筛选器 (Screener)

#### 2.1.1 基础筛选条件
| 类别 | 条件项 | 说明 |
|------|--------|------|
| 市场 | 沪市/深市/创业板/科创板/北交所 | 单选或多选 |
| 行业 | 一级/二级行业分类 | 下拉选择 |
| 概念 | 热门概念板块 | 多选 |
| 地域 | 省份/城市 | 下拉选择 |

#### 2.1.2 技术指标筛选
| 指标 | 参数 | 运算符 |
|------|------|--------|
| 最新价 | 元 | >, <, =, >=, <=, 区间 |
| 涨跌幅 | % | >, <, =, >=, <=, 区间 |
| 换手率 | % | >, <, >=, <= |
| 成交量 | 手 | >, <, >=, <= |
| 成交额 | 万元 | >, <, >=, <= |
| 市盈率(PE) | 倍 | >, <, >=, <=, 区间 |
| 市净率(PB) | 倍 | >, <, >=, <=, 区间 |
| 总市值 | 亿元 | >, <, >=, <= |
| 流通市值 | 亿元 | >, <, >=, <= |

#### 2.1.3 技术指标筛选
| 指标 | 说明 |
|------|------|
| MACD | 金叉/死叉/多头排列/空头排列 |
| KDJ | K值、D值、J值范围，金叉/死叉 |
| RSI | 6日/12日/24日 RSI 值范围 |
| 均线 | 5日/10日/20日/60日均线，多头排列/空头排列/突破 |
| 布林带 | 触及上轨/下轨/中轨，开口/缩口 |
| 成交量 | 放量/缩量，量比范围 |

#### 2.1.4 资金流向筛选
| 指标 | 说明 |
|------|------|
| 主力净流入 | 当日主力净流入金额范围 |
| 主力净流入占比 | 主力净流入占成交额比例 |
| 超大单净流入 | 超大单净流入金额 |
| 大单净流入 | 大单净流入金额 |
| 连续净流入天数 | 主力连续净流入天数 |

#### 2.1.5 财务指标筛选
| 指标 | 说明 |
|------|------|
| 营收增长率 | 同比/环比增长 |
| 净利润增长率 | 同比/环比增长 |
| ROE | 净资产收益率 |
| ROA | 总资产收益率 |
| 毛利率 | 销售毛利率 |
| 净利率 | 销售净利率 |
| 负债率 | 资产负债率 |
| 流动比率 | 流动资产/流动负债 |
| 速动比率 | (流动资产-存货)/流动负债 |
| 利息保障倍数 | 息税前利润/利息费用 |
| 营业周期 | 存货周转天数+应收账款周转天数 |
| 存货周转率 | 营业成本/平均存货 |
| 应收账款周转率 | 营业收入/平均应收账款 |
| 总资产周转率 | 营业收入/平均总资产 |
| 经营现金流/净利润 | 经营现金流净额/净利润 |
| 自由现金流 | 经营现金流-资本支出 |
| 股息率 | 年度分红/当前股价 |
| 分红率 | 分红金额/净利润 |
| 连续分红年数 | 连续分红年数 |

### 2.1.6 价值投资指标筛选
| 指标 | 说明 |
|------|------|
| PEG | 市盈率/盈利增长率，<1 可能低估 |
| 市现率(PCF) | 股价/每股现金流 |
| 市销率(PS) | 总市值/营业收入 |
| 企业价值倍数(EV/EBITDA) | 企业价值/息税折旧摊销前利润 |
| 账面市值比 | 每股净资产/股价 |
| 盈利收益率 | 每股收益/股价 |
| 股东权益比率 | 股东权益/总资产 |
| 有形资产净值 | 净资产-无形资产-商誉 |
| 研发投入占比 | 研发费用/营业收入 |
| 资本支出占比 | 资本支出/营业收入 |
| 近5年ROE均值 | 长期盈利能力 |
| 近5年营收复合增长率 | 长期成长性 |
| 近5年净利润复合增长率 | 长期盈利增长 |
| 机构持股比例 | 机构持仓占比 |
| 北向资金持股 | 沪深港通持股 |
| 股东户数变化 | 筹码集中度变化 |

### 2.2 股票详情页 (Stock Detail)

#### 2.2.1 K线图表
- 时间周期: 日线/周线/月线
- 技术指标叠加: MA、MACD、KDJ、RSI、布林带
- 成交量柱状图
- 缩放、平移、十字光标

#### 2.2.2 基本信息面板
- 股票名称、代码、所属行业
- 实时行情数据（价格、涨跌幅、成交量等）
- 五档盘口数据

#### 2.2.3 技术指标面板
- 各技术指标当前数值
- 指标状态（超买/超卖/金叉/死叉等）

#### 2.2.4 资金流向面板
- 当日资金流向饼图（超大单/大单/中单/小单）
- 近5日/10日/20日资金流向趋势图
- 主力净流入/流出排名

#### 2.2.5 财务数据面板
- 主要财务指标一览
- 近期财报数据（季报/年报）
- 财务指标趋势图（近5年）
- 杜邦分析（ROE拆解）
- 同行业对比分析
- 财务健康度评分

#### 2.2.6 价值投资面板
- 估值指标一览（PE/PB/PEG/PCF/PS）
- 历史估值百分位（近5年/近10年）
- 估值合理性评估
- 内在价值估算（DCF/格雷厄姆公式）
- 安全边际分析
- 股息分红记录
- 股东结构分析

### 2.3 自选股管理 (Watchlist)

#### 2.3.1 自选股分组
- 支持创建多个分组（如"重点关注","短线操作","长线持有"）
- 股票可归属于多个分组

#### 2.3.2 自选股列表
- 表格展示：代码、名称、最新价、涨跌幅、换手率等
- 实时刷新（可配置刷新间隔）
- 点击跳转详情页
- 支持排序、筛选

#### 2.3.3 预警功能
- 价格预警（突破/跌破指定价格）
- 涨跌幅预警（单日涨跌幅超过阈值）
- 成交量预警（放量/缩量）
- 技术指标预警（金叉/死叉等）
- 预警通知：弹窗提示 + 声音提醒

### 2.4 市场概览 (Market Overview)

#### 2.4.1 大盘指数
- 上证指数、深证成指、创业板指、科创50
- 指数走势图
- 涨跌家数统计

#### 2.4.2 板块热点
- 行业板块涨幅排行
- 概念板块涨幅排行
- 板块资金流向

#### 2.4.3 涨跌分布
- 涨跌家数饼图
- 涨跌停家数统计

#### 2.4.4 资金流向总览
- 市场整体资金流向
- 北向资金流向

#### 2.4.5 估值概览
- 市场整体PE/PB水平
- 主要指数估值百分位
- 行业估值分布
- 破净股统计
- 高股息股票池

### 2.5 策略回测 (Backtest)

#### 2.5.1 策略定义
- 基于筛选条件创建策略
- 支持多条件组合
- 买入/卖出条件设置
- 支持技术策略、价值策略、混合策略

#### 2.5.2 回测参数
- 回测时间范围
- 初始资金
- 仓位管理（固定金额/固定比例/凯利公式）
- 交易费率设置
- 滑点设置
- 再平衡周期（价值投资定期调仓）

#### 2.5.3 回测结果
- 收益曲线图
- 交易记录列表
- 绩效指标：总收益率、年化收益率、最大回撤、夏普比率、胜率等
- 年度收益分布
- 月度收益热力图
- 导出回测报告

### 2.6 价值投资分析工具

#### 2.6.1 股票对比工具
- 多股票财务指标对比
- 估值水平对比
- 盈利能力对比
- 成长性对比

#### 2.6.2 估值计算器
- DCF现金流折现估值
- 格雷厄姆估值公式
- 彼得林奇PEG估值
- 历史估值区间参考

#### 2.6.3 财报分析
- 财务报表可视化
- 财务指标趋势分析
- 异常指标预警
- 同行业排名

#### 2.6.4 优质股票池
- 高ROE股票池（连续5年ROE>15%）
- 低估值股票池（PE<行业均值，PB<2）
- 高股息股票池（股息率>3%，连续分红>5年）
- 成长股池（营收/净利润增速>20%）
- 白马股池（大盘蓝筹，业绩稳定）

### 2.7 数据管理

#### 2.7.1 数据更新
- 手动更新/自动更新
- 更新进度显示
- 增量更新机制
- 财报数据定期更新

#### 2.7.2 数据导出
- 筛选结果导出 Excel
- 自选股导出/导入
- 回测报告导出
- 财务数据导出
- 估值分析报告导出

## 4. 技术架构

### 4.1 技术栈
- **编程语言**: Python 3.8+
- **GUI框架**: PyQt6
- **数据处理**: pandas, numpy
- **数据可视化**: pyqtgraph, matplotlib
- **数据库**: SQLite (SQLAlchemy < 2.0)
- **数据源**: tushare, baostock

### 4.2 模块架构

## 5. 数据源设计

### 5.1 数据源配置
- 支持多个数据源（tushare、baostock）
- API Key 配置管理
- 数据源优先级设置
- 自动切换/故障转移

### 5.2 数据缓存
- 本地 SQLite 数据库缓存
- 缓存策略：股票列表(每日)、日线数据(本地)、实时数据(内存)
- 数据过期自动更新

### 5.3 数据更新策略
| 数据类型 | 更新频率 | 存储方式 |
|----------|----------|----------|
| 股票列表 | 每日 | SQLite |
| 日线数据 | 每日收盘后 | SQLite |
| 实时行情 | 实时/定时 | 内存+SQLite |
| 财务数据 | 季报发布时 | SQLite |
| 资金流向 | 实时/定时 | 内存 |
| 估值历史数据 | 每日收盘后 | SQLite |

## 6. 非功能需求

### 6.1 性能要求
- 筛选全市场股票响应时间 < 3秒
- 界面切换流畅，无卡顿
- 支持同时监控 100+ 自选股
- 内存占用 < 500MB

### 6.2 可靠性
- 数据源异常时自动重试/切换
- 本地缓存保证离线可用
- 关键操作有确认提示

### 6.3 易用性
- 界面简洁直观
- 筛选条件可保存/加载
- 支持快捷键操作
- 提供操作提示和帮助

### 6.4 扩展性
- 模块化设计，便于新增数据源
- 策略系统支持自定义扩展
- 指标计算可插拔
