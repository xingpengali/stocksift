# 表现层设计文档

## 1. 概述

表现层负责用户界面展示和交互，基于 PyQt6 实现。采用页面+组件的架构，支持主题切换和响应式布局。

## 2. 整体架构

```
ui/
├── main_window.py        # 主窗口
├── theme_manager.py      # 主题管理
├── base_page.py          # 页面基类
├── pages/                # 页面
│   ├── market_overview.py
│   ├── screener_page.py
│   ├── stock_detail.py
│   ├── watchlist.py
│   ├── backtest_page.py
│   └── value_investing_page.py
├── widgets/              # 可复用组件
│   ├── kline_chart.py
│   ├── stock_table.py
│   ├── filter_panel.py
│   └── ...
└── dialogs/              # 对话框
    ├── settings_dialog.py
    ├── alert_dialog.py
    └── ...
```

## 4. 页面基类 (base_page)

### 4.1 职责
- 定义页面生命周期
- 提供通用方法
- 处理页面切换事件

### 4.2 接口定义

```python
class BasePage(QWidget):
    page_id: str = ""           # 页面标识
    page_name: str = ""         # 页面名称
    
    def __init__(self, parent=None)
    
    # 生命周期
    def on_enter()              # 进入页面时调用
    def on_leave()              # 离开页面时调用
    def on_refresh()            # 刷新页面
    
    # 数据更新
    def update_data(data: Dict)
    
    # 消息处理
    def show_message(message: str)
    def show_error(error: str)
    def show_loading(show: bool)
```

## 5. 页面详细设计

### 5.1 市场概览页 (market_overview)

```python
class MarketOverviewPage(BasePage):
    page_id = "market"
    page_name = "市场概览"
    
    def __init__(self)
        # 布局
        self.index_panel      # 大盘指数
        self.sector_panel     # 板块热点
        self.statistics_panel # 涨跌统计
        self.flow_panel       # 资金流向
```

**布局分区：**
- 顶部：大盘指数卡片（上证、深证、创业、科创）
- 左侧：板块涨幅排行
- 右侧：涨跌分布图
- 底部：资金流向、估值概览
