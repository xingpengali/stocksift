# 分析层设计文档

## 1. 概述

分析层提供股票分析相关的计算能力，包括技术分析、基本面分析、估值分析、资金流向分析等。该层只负责计算，不涉及数据存储和UI展示。

## 2. 模块结构

```
analysis/
├── technical.py          # 技术分析
├── fundamental.py        # 基本面分析
├── valuation.py          # 估值分析
├── capital_flow.py       # 资金流向分析
├── financial_health.py   # 财务健康度
└── sentiment.py          # 情绪分析
```

## 3. 技术分析 (technical)

### 3.1 功能职责
- 技术指标计算（MACD、KDJ、RSI、均线、布林带）
- 技术信号识别（金叉、死叉、突破）

### 3.2 核心接口


## 4. 基本面分析 (fundamental)

### 4.1 功能职责
- 财务指标计算与提取
- 同行业对比分析
- 财务趋势分析

### 4.2 核心接口

```python
class FundamentalAnalyzer:
    def __init__(self, financial_data: List[Dict])
    
    # 盈利能力
    def profitability() -> Dict:
        # ROE, ROA, 毛利率, 净利率
    
    # 成长能力
    def growth() -> Dict:
        # 营收增长率, 净利润增长率
    
    # 运营效率
    def efficiency() -> Dict:
        # 存货周转率, 应收账款周转率, 总资产周转率
    
    # 偿债能力
    def solvency() -> Dict:
        # 流动比率, 速动比率, 资产负债率, 利息保障倍数
    
    # 现金流
    def cashflow() -> Dict:
        # 经营现金流, 自由现金流, 现金流/净利润比
    
    # 同行业对比
    def compare_with_industry(industry_data: List[Dict]) -> Dict:
        # 各项指标行业排名
    
    # 杜邦分析
    def dupont_analysis() -> Dict:
        # ROE = 净利率 × 总资产周转率 × 权益乘数
```

## 5. 估值分析 (valuation)

### 5.1 功能职责
- 各类估值指标计算
- 历史估值百分位
- 内在价值估算

### 5.2 核心接口

```python
class ValuationAnalyzer:
    def __init__(self, stock_code: str, quote: Quote, financial: Financial)
    
    # 估值指标
    def pe_ratio() -> Dict:
        # PE_TTM, PE_LYR, 历史百分位
    
    def pb_ratio() -> Dict:
        # PB, 历史百分位
    
    def peg_ratio() -> float:
        # PE / 盈利增长率
    
    def pcf_ratio() -> float:
        # 市现率
    
    def ps_ratio() -> float:
        # 市销率
    
    def ev_ebitda() -> float:
        # 企业价值倍数
    
    # 历史估值百分位
    def historical_percentile(years=5) -> Dict:
        # PE/PB在过去N年的百分位
    
    # 内在价值估算
    def dcf_valuation(
        growth_rate: float,      # 预期增长率
        discount_rate: float,    # 折现率
        terminal_growth: float   # 永续增长率
    ) -> Dict:
        # DCF估值结果
    
    def graham_valuation() -> float:
        # 格雷厄姆估值: EPS × (8.5 + 2g)
    
    # 估值评估
    def valuation_assessment() -> Dict:
        # 综合评估: 低估/合理/高估
```
