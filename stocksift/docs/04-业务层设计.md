# 业务层设计文档

## 1. 概述

业务层实现核心业务逻辑，包括股票筛选、策略管理、回测引擎、预警系统等。该层协调数据层和分析层，为UI层提供业务接口。

## 2. 模块结构

```
core/
├── screener.py           # 筛选引擎
├── strategy.py           # 策略管理
├── backtest.py           # 回测引擎
├── alert_engine.py       # 预警引擎
└── data_fetcher.py       # 数据获取协调器
```

## 3. 筛选引擎 (screener)

### 3.1 功能职责
- 多维度条件组合筛选
- 动态SQL构建
- 结果排序与分页
- 筛选条件持久化

### 3.2 筛选条件定义

```python
@dataclass
class FilterCondition:
    field: str               # 字段名
    operator: str            # >, <, =, >=, <=, between, in
    value: Any               # 值
    value2: Any = None       # 范围第二个值
    logic: str = "AND"       # AND/OR

@dataclass
class FilterGroup:
    name: str
    conditions: List[FilterCondition]
    logic: str = "AND"
```

### 3.3 核心接口

```python
class ScreenerEngine:
    def __init__(self, db_session)
    
    # 执行筛选
    def screen(
        self,
        conditions: List[FilterCondition],
        order_by: str = None,
        order_desc: bool = True,
        page: int = 1,
        page_size: int = 50
    ) -> ScreenResult
    
    # 构建查询
    def build_query(conditions: List[FilterCondition]) -> Query
    
    # 获取可用字段
    def get_available_fields() -> List[Dict]
    
    # 保存/加载筛选条件
    def save_filter(name: str, conditions: List[FilterCondition])
    def load_filter(name: str) -> List[FilterCondition]
    def list_saved_filters() -> List[str]

@dataclass
class ScreenResult:
    total: int                    # 总数
    page: int                     # 当前页
    page_size: int                # 每页数量
    data: List[Dict]              # 结果数据
    execution_time: float         # 执行时间
```

### 3.4 支持的筛选字段

| 类别 | 字段 |
|------|------|
| 基础 | code, name, exchange, market_type, industry |
| 行情 | price, change_pct, volume, amount, turnover |
| 估值 | pe_ttm, pb, total_mv, float_mv |
| 技术 | macd_signal, kdj_signal, rsi6, ma_alignment |
| 资金 | main_inflow, main_inflow_pct |
| 财务 | roe, revenue_growth, profit_growth, gross_margin |

## 4. 策略管理 (strategy)

### 4.1 功能职责
- 策略定义与管理
- 策略与筛选条件关联
- 策略执行调度

### 4.2 策略定义

```python
@dataclass
class Strategy:
    id: str
    name: str
    description: str
    
    # 筛选条件
    entry_conditions: List[FilterCondition]   # 买入条件
    exit_conditions: List[FilterCondition]    # 卖出条件
    
    # 参数
    parameters: Dict                          # 策略参数
    
    # 元数据
    created_at: datetime
    updated_at: datetime
    is_active: bool
```

### 4.3 核心接口

```python
class StrategyManager:
    def create(strategy: Strategy) -> str
    def update(strategy: Strategy)
    def delete(strategy_id: str)
    def get(strategy_id: str) -> Strategy
    def list_all() -> List[Strategy]
    
    # 执行策略筛选
    def run(strategy_id: str) -> ScreenResult
    
    # 策略回测（调用BacktestEngine）
    def backtest(strategy_id: str, params: BacktestParams) -> BacktestResult
```

## 5. 回测引擎 (backtest)

### 5.1 功能职责
- 历史数据回测
- 交易模拟
- 绩效指标计算

### 5.2 回测参数

```python
@dataclass
class BacktestParams:
    strategy: Strategy           # 策略
    start_date: date             # 开始日期
    end_date: date               # 结束日期
    initial_capital: float       # 初始资金
    position_size: float         # 仓位比例
    commission_rate: float       # 手续费率
    slippage: float              # 滑点
    rebalance_period: int        # 再平衡周期（天）
```

### 5.3 核心接口

```python
class BacktestEngine:
    def __init__(self, data_fetcher: DataFetcher)
    
    # 执行回测
    def run(params: BacktestParams) -> BacktestResult
    
    # 获取回测结果
    def get_result(backtest_id: str) -> BacktestResult

@dataclass
class BacktestResult:
    # 收益指标
    total_return: float          # 总收益率
    annualized_return: float     # 年化收益率
    benchmark_return: float      # 基准收益率
    
    # 风险指标
    max_drawdown: float          # 最大回撤
    volatility: float            # 波动率
    sharpe_ratio: float          # 夏普比率
    sortino_ratio: float         # 索提诺比率
    
    # 交易统计
    total_trades: int            # 总交易次数
    winning_trades: int          # 盈利次数
    losing_trades: int           # 亏损次数
    win_rate: float              # 胜率
    profit_factor: float         # 盈亏比
    
    # 详细记录
    equity_curve: List[Dict]     # 权益曲线
    trades: List[TradeRecord]    # 交易记录
    monthly_returns: List[Dict]  # 月度收益
```

## 6. 预警引擎 (alert_engine)

### 6.1 功能职责
- 实时监控股票数据
- 预警条件判断
- 预警通知推送

### 6.2 预警规则定义

```python
@dataclass
class AlertRule:
    id: str
    name: str
    code: str                    # 监控股票
    
    # 触发条件
    condition_type: str          # price, change, volume, indicator
    operator: str                # above, below, cross_up, cross_down
    threshold: float             # 阈值
    
    # 通知设置
    notify_popup: bool           # 弹窗通知
    notify_sound: bool           # 声音提醒
    
    # 状态
    is_active: bool
    created_at: datetime
```

### 6.3 核心接口

```python
class AlertEngine:
    def __init__(self, quote_cache: QuoteCache)
    
    # 规则管理
    def add_rule(rule: AlertRule) -> str
    def update_rule(rule: AlertRule)
    def delete_rule(rule_id: str)
    def get_rules(code: str = None) -> List[AlertRule]
    
    # 启动/停止监控
    def start()
    def stop()
    
    # 检查触发（定时调用）
    def check_alerts()
    
    # 获取预警历史
    def get_alert_history(rule_id: str = None) -> List[AlertRecord]

@dataclass
class AlertRecord:
    id: str
    rule_id: str
    code: str
    triggered_at: datetime
    trigger_value: float
    message: str
    is_read: bool
```

## 7. 数据获取协调器 (data_fetcher)

### 7.1 功能职责
- 统一数据获取接口
- 数据源协调与故障转移
- 数据缓存与更新

### 7.2 核心接口

```python
class DataFetcher:
    def __init__(self, source_manager: DataSourceManager)
    
    # 股票数据
    def get_stock_list(force_update: bool = False) -> List[Stock]
    def get_stock_basic(code: str) -> Stock
    
    # 行情数据
    def get_quotes(codes: List[str]) -> List[Quote]
    def get_kline(code: str, period: str, count: int) -> List[Kline]
    
    # 财务数据
    def get_financial(code: str) -> Financial
    def get_financial_history(code: str, years: int) -> List[Financial]
    
    # 更新数据
    def update_stock_list()
    def update_quotes(codes: List[str])
    def update_kline(code: str)
```
