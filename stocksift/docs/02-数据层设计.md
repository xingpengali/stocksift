# 数据层设计文档

## 1. 概述

数据层负责数据的持久化存储和外部数据接入，包括本地数据库管理、数据模型定义、数据源适配器等。

## 2. 数据库管理 (models/database)

### 2.1 技术选型
- **数据库**: SQLite（本地文件，零配置）
- **ORM**: SQLAlchemy < 2.0.0
- **连接池**: SQLAlchemy内置连接池

### 2.2 核心组件

```python
# 数据库管理器
class DatabaseManager:
    def init_db()              # 初始化数据库（建表）
    def get_session()          # 获取会话
    def session_scope()        # 上下文管理器
    def close()                # 关闭连接

# 全局实例
db_manager = DatabaseManager()
```

### 2.3 连接池配置

```python
engine = create_engine(
    f"sqlite:///{db_path}",
    pool_size=5,
    max_overflow=10,
    pool_recycle=3600,
    connect_args={"check_same_thread": False}
)
```

### 3.2 核心模型设计

#### Stock（股票基本信息）

```python
class Stock(Base):
    __tablename__ = "stocks"
    
    id = Column(Integer, primary_key=True)
    code = Column(String(10), unique=True, index=True)  # 股票代码
    name = Column(String(50))                           # 股票名称
    exchange = Column(String(10))                       # 交易所
    market_type = Column(String(10))                    # 市场类型
    industry_code = Column(String(10))                  # 行业代码
    industry_name = Column(String(50))                  # 行业名称
    concept_list = Column(JSON)                         # 概念板块
    province = Column(String(20))                       # 省份
    city = Column(String(20))                           # 城市
    list_date = Column(Date)                            # 上市日期
    total_shares = Column(BigInteger)                   # 总股本
    float_shares = Column(BigInteger)                   # 流通股本
```

#### Quote（实时行情）

```python
class Quote(Base):
    __tablename__ = "quotes"
    
    id = Column(Integer, primary_key=True)
    code = Column(String(10), index=True)
    price = Column(DECIMAL(10, 2))           # 最新价
    pre_close = Column(DECIMAL(10, 2))       # 昨收
    open = Column(DECIMAL(10, 2))            # 开盘价
    high = Column(DECIMAL(10, 2))            # 最高价
    low = Column(DECIMAL(10, 2))             # 最低价
    change = Column(DECIMAL(10, 2))          # 涨跌额
    change_pct = Column(DECIMAL(6, 2))       # 涨跌幅%
    volume = Column(BigInteger)              # 成交量
    amount = Column(DECIMAL(16, 2))         # 成交额
    pe_ttm = Column(DECIMAL(10, 2))          # 市盈率
    pb = Column(DECIMAL(10, 2))              # 市净率
    total_mv = Column(DECIMAL(16, 2))       # 总市值
    float_mv = Column(DECIMAL(16, 2))       # 流通市值
    turnover = Column(DECIMAL(6, 2))         # 换手率
    quote_time = Column(DateTime)            # 行情时间
```

## 4. 数据源适配器 (datasource/)

### 4.1 架构设计

```
datasource/
├── base_adapter.py           # 抽象基类
├── tushare_adapter.py        # Tushare实现
├── baostock_adapter.py       # Baostock实现
└── data_source_manager.py    # 数据源管理器
```

### 4.2 抽象接口

```python
class BaseDataAdapter(ABC):
    # 连接管理
    def connect() -> bool
    def disconnect()
    def is_connected() -> bool
    
    # 股票数据
    def get_stock_list() -> List[Dict]
    def get_stock_basic(code: str) -> Optional[Dict]
    
    # 行情数据
    def get_daily_quotes(codes: List[str]) -> List[Dict]
    def get_realtime_quotes(codes: List[str]) -> List[Dict]
    def get_kline_data(code, start, end, period) -> List[Dict]
    
    # 财务数据
    def get_financial_data(code, report_type) -> Optional[Dict]
    def get_income_statement(code, start, end) -> List[Dict]
    def get_balance_sheet(code, start, end) -> List[Dict]
    def get_cash_flow(code, start, end) -> List[Dict]
    
    # 市场数据
    def get_index_list() -> List[Dict]
    def get_industry_list() -> List[Dict]
    def get_concept_list() -> List[Dict]
    
    # 资金流向
    def get_capital_flow(code, date) -> Optional[Dict]
```

### 4.3 数据源管理器

class DataSourceManager:

## 5. 数据更新策略

| 数据类型 | 更新频率 | 存储方式 | 来源 |
|----------|----------|----------|------|
| 股票列表 | 每日 | SQLite | Tushare/Baostock |
| 日线数据 | 每日收盘后 | SQLite | Tushare/Baostock |
| 实时行情 | 5秒 | 内存+SQLite | Tushare |
| 财务数据 | 季报发布时 | SQLite | Tushare |
| 资金流向 | 实时 | 内存 | Tushare |
| 估值数据 | 每日收盘后 | SQLite | 计算生成 |
